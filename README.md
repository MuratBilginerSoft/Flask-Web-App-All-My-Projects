# Flask Web App - All My Projects

Bu repo, Python Flask framework ile geliştirilmiş 8 adet web uygulaması içermektedir. Projeler temel Flask kullanımından FullStack template yapılarına, REST API tasarımından veritabanı yönetimine kadar geniş bir yelpazede yer almaktadır.

## Proje Listesi

| # | Proje | Açıklama |
|---|-------|----------|
| 1 | Flask_01_Hello_World | Flask ile Hello World uygulaması |
| 2 | Flask_02_App_Template | Flask uygulama şablonu |
| 3 | Flask_03_Rest_API_Template | Flask REST API şablonu |
| 4 | Flask_04_SQLAlchemy_Database_Orchestration | SQLAlchemy veritabanı yönetimi |
| 5 | Flask_05_FullStack_Template | Flask FullStack şablonu |
| 6 | Flask_06_Azure_Flask_App | Azure üzerinde Flask uygulaması |
| 7 | Flask_07_Brainy_Flask_Architecture | Brainy Flask mimari yapısı |
| 8 | Flask_08_Mail_Send | Flask ile mail gönderme |

## Kapsanan Konular

### Web Geliştirme
- Flask temel yapı ve routing
- Template engine (Jinja2)
- REST API tasarımı
- FullStack uygulama mimarisi

### Veritabanı
- SQLAlchemy ORM
- Database migration (DbInit, DbMigrade, DbCreate)
- Model tasarımı

### Dağıtım ve Servisler
- Azure deployment
- Mail gönderme servisi
- Modüler mimari yapılar

## Gereksinimler

- Python 3.x
- Flask (`pip install flask`)
- SQLAlchemy (`pip install flask-sqlalchemy`)

## Kullanım

Her proje klasörü bağımsız olarak çalıştırılabilir:

```bash
cd Flask_01_Hello_World
pip install -r requirements.txt
python app.py
```
